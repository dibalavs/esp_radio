Чеклист проверки платы:

Схема:
1) Отдельные классы цепей на питание (линии толще чем данные)
2) Кварц подключен к MCU
3) SWD подключен
4) Конденсаторы по питанию перед микросхемами.
5) Защита от переполюсовки по питанию.
6) Добавлены Test Pins для питания, GND и нужных сигналов.


Разводка:
1) Есть крепежные отверстия
2) Габариты платы подхдят для корпуса
3) Шелкография номеров компонентов
4) Шелкография номиналов компонентов
5) Шелкография подписей к разъемам.
6) Первые выводы микросхем промаркированы
7) Свободное место на плате залито GND
8) + питания на разъеме промаркирован
9) Для высокотоковых цепей используются контактные площадки с отверстием >= 1.2 мм диаметра.
10) Добавлен копирайт.
